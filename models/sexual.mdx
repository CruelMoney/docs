## Labels

| Label            | Description                                |
| ---------------- | ------------------------------------------ |
| **SEXUAL**       | Explicitly sexual content.                 |
| **SEXUAL_MINOR** | Only triggered if the subject is under 18. |
| **NEUTRAL**      | Nothing sexual was detected.               |
