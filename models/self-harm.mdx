> Model key: `self_harm`

```json Response Example:
{
  "label": "SELF_HARM_INTENT",
  "score": 0.99966,
  "label_scores": {
    "SELF_HARM_INTENT": 0.99966,
    "SELF_HARM": 0.99931,
    "SELF_HARM_DIRECTION": 0.59209,
    "NEUTRAL": 0.00034
  }
}
```

## Labels

| Label                   | Description                                                               |
| ----------------------- | ------------------------------------------------------------------------- |
| **SELF_HARM**           | Content referencing self harm like suicide, cutting, or eating disorders. |
| **SELF_HARM_INTENT**    | The author is expressing an intent to self harm.                          |
| **SELF_HARM_DIRECTION** | The author is giving directions for self harm.                            |
| **NEUTRAL**             | No references to self harm.                                               |
