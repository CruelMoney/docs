
## Labels

| Label                       | Description                                                               |
| --------------------------- | ------------------------------------------------------------------------- |
| **DISCRIMINATION**          | Discrimination against any group.                                         |
| **DISCRIMINATION_MINORITY** | Discrimination of religion, ethnicity, sexual orientation, disabled, etc. |
| **NEUTRAL**                 | Neutral content.                                                          |
