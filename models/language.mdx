> Model key: `language`

<Check>Does not count against moderation quota.</Check>

Detect what language a text is written in.
Probabilistically detects over 160 languages. We're not listing all the languages here, but if you need a comprehensive list, please message us.

## Response

<Note>
  The language classifier has a different response format than the other
  classifiers.
</Note>

<ResponseField name="code" type="string">
  The ISO 639 of the language. Returns null if the analyzer fails.
</ResponseField>
<ResponseField name="label" type="string">
  An UPPERCASE name of the language. Returns null if the analyzer fails.
</ResponseField>
<ResponseField name="score" type="int" required>
  0-1 score with 1 meaning a high probability of being correct. Returns 0 on
  fails.
</ResponseField>
<ResponseField name="reliable" type="int" required>
  The analyzer is reasonably confident about this guess, e.g. the score is high.
</ResponseField>
<ResponseField name="error" type="string">
  If the analyzer failed an error will be returned.
</ResponseField>

```json Response Example:
{
  "code": "en",
  "label": "ENGLISH",
  "score": 0.96,
  "reliable": true
}
```
